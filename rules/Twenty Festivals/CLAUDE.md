# L5R Twenty Festivals — Codified Rules
 
Authoritative rules reference for the **Legend of the Five Rings Collectible Card Game: Twenty Festivals** edition. Distilled verbatim from the three source PDFs in this directory:
 
- `Comprehensive Rules for L5R_ Twenty Festivals - AEG Rules.pdf` (primary authoritative source)
- `Rulebook-L5R-20F-basic_rulebook.pdf`
- `Rulebook-L5R-20F-adv_rulebook.pdf`
## Source-of-truth policy
 
Per Cardinal Rule 3, **Comprehensive Rules wins** any contradiction with the Basic or Advanced Rulebook. Rulebook-only content (e.g. the step-by-step Dueling walkthrough) is folded in as elaboration. Genuine contradictions are flagged `[CONFLICT: source A vs source B]`.

## Writing conventions
 
- Every section uses `##`/`###` headings; base rules are stated first and **Exceptions:** are listed explicitly.
- Numeric values are quoted exactly (e.g. "Maximum hand size: 8").
- Cross-references are by file + section (e.g. "see: `08-turn-structure.md` §Attack Phase").
- Glossary entries are formatted `**Term** — definition.`
- No inferred rules — anything not in the source PDFs is omitted.
- No flavor text, no strategy advice, no narrative.
